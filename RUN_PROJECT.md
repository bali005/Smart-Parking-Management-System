# Run Project

Open two PowerShell windows in the project root and run:

```powershell
.\start-backend.ps1
```

```powershell
.\start-frontend.ps1
```

App URLs:

- Frontend: `http://localhost:5174`
- Backend: `http://localhost:7070`

Backend notes:

- Requires local MySQL running on `localhost:3306`
- Uses database `parking_db`
- Uses username `root`
- Uses password `root`
